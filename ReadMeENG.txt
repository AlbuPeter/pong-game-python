1. Preparation
  1.1 Download and Install Python from Microsoft Store
  
    Open the Microsoft Store on your computer.
    
    Search for: Python 3.12 (or the latest available version).
    
    Click the Install button.
    
    This will automatically install Python and the pip package manager, and add Python to your system PATH (no extra configuration needed).
  
  1.2 Verification
  
    Open a Command Prompt (cmd) and type:
    
      python --version
  
  
    If it shows a version (e.g., Python 3.12.3), the installation was successful.
  
  1.3 Install pygame
  
    The Pong game requires the pygame library. Install it with:
    
      python -m pip install pygame
    
    
    If no errors appear, pygame is ready to use.

2. Pong Game Source Code

  The game code is available on GitHub in a file called pong.py.
  Download it or copy it to your computer, and save it in a folder of your choice.

3. Running the Game

  Open a Command Prompt in the folder where you saved pong.py.
  Example:
  
    cd <path-to-your-folder>
  
  
  Start the program with:
  
    python pong.py
